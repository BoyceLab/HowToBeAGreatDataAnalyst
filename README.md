# How to Be a Great Data Analyst — Project Toolkit
Companion materials for the book How to Be a Great Data Analyst: Eight Practical Habits Learned From My 30 Years in the Field

This repository contains worksheets, templates, and resources that accompany the book:

**How to Be a Great Data Analyst: Eight Practical Habits Learned From My 30 Years in the Field**  
by Dr. Danielle Boyce

The materials here are designed to help analysts, students, researchers, and patient advocacy organizations apply the principles described in the book to real-world projects.

---

## About the Toolkit

Real-world data analysis involves more than writing code or running statistical models. Successful projects require careful planning, documentation, communication, and attention to data quality.

The tools in this repository support those practices.

These materials can be used for:

- research projects
- student research internships
- academic coursework
- patient registry analysis
- collaborative research teams

---

## How to Use These Materials

1. Download the worksheet templates.
2. Adapt them to your research environment or project needs.
3. Use them to structure analytical planning, documentation, and communication.

These tools are intentionally simple and flexible so that they can be used in a wide range of research settings.
